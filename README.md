# 使用方式一
mac:cmd+shift+p.  
window:ctri+shift+p.  
打开搜素框  
输入关键字搜索，例如："电子邮箱"  
![image](https://user-images.githubusercontent.com/8264787/146726350-58166bf9-7f7f-4685-86aa-f055d0c0b914.png)
